Developing for the Web

This website will provide you with information on the following:

  1. Using HTML to produce the pages of your website
  2. Implementing CSS to format your web pages
  3. How search engines work
  4. Improving search results

Hypertext Markup Language (HTML) and Cascading Style Sheets (CSS) are used to create and format pages of a website. CSS allows you to make formatting changes to your website in one document. The 'intsructions' in this document are then applied to your web pages.
